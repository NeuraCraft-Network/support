## Getting Started

Welcome to **NeuraCraft**!  
This guide will help you get started and understand what to do when joining the server for the first time.

---

## Server Information

### Minecraft Bedrock Edition
- **Server Address:** `mc.neuracraft.my.id`
- **Port:** `19162`
- **Supported Version:** **Minecraft Bedrock 1.20.x – Latest**

> ⚠️ This server is **BEDROCK EDITION ONLY**.  
> Java Edition is **not supported**.

---

## How to Join
1. Open **Minecraft Bedrock Edition**
2. Go to **Play → Servers**
3. Add a new server
4. Enter the server address and port
5. Save and join the server

---

## First Time Joining
When you join the server for the first time:

- You will be guided through an **in-game tutorial**
- Please **read and follow the tutorial carefully**
- The tutorial explains:
  - Basic gameplay mechanics
  - Important commands
  - Server features
  - Rules you must follow

➡ **Do not skip the tutorial**, as it contains essential information for new players.

---

## Important Commands
Commonly used commands include:
- `/help` – View available commands
- `/hub` – Return to spawn
- `/rules` – View server rules
- `/discord` – Get the official Discord link (if available)
