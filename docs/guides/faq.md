## Frequently Asked Questions (FAQ)

Below are the most frequently asked questions regarding the **NeuraCraft** Minecraft server.

---

### What Minecraft version does this server support?
This server supports **Minecraft Bedrock Edition only**,  
version **1.20.x to the latest available version**.

---

### Does this server support Java Edition?
No.  
This server is **Bedrock Edition only** and does **not support Java Edition**.

---

### How do I join the server?
- **Server Address:** `mc.neuracraft.my.id`
- **Port:** `19132/19162`

A full guide is available here:  
[Getting Started](/guides/getting-started)

---

### Is this server free to play?
Yes.  
The server is **free to play**.

Optional purchases are available to support server development, but they are **not required**.

---

### Is this server Pay-to-Win (P2W)?
The server includes **some paid features**, however:
- Paid features are not excessively powerful
- Free-to-play players can still compete fairly
- Gameplay balance is actively maintained

---

### Is griefing allowed?
**It depends on the game mode.**  
Griefing is only allowed in **specific modes or areas** where it is clearly stated as allowed.

If an area or mode does not explicitly allow griefing, it is considered **not allowed**.

---

### Are traps allowed?
Yes, but **only in modes or areas where traps are explicitly permitted**.

Traps that abuse bugs or cause excessive server lag are not allowed.

---

### Is flying allowed?
Flying is **only allowed** if it is provided by:
- Official server features
- Premium ranks
- Server gameplay mechanics

Flying using **hack clients or third-party software is strictly prohibited**.

---

### What should I do if I find a bug?
Do not abuse it.  
Please **report the bug to staff** through official server channels.
